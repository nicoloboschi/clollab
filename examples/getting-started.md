# Getting Started with clollab

clollab is an in-browser markdown editor powered by Claude Code. You can select any text and leave a comment — Claude will apply the change directly to the file.

## How to use it

1. Open a markdown file from the sidebars
2. Select any piece of text you want to change
3. Click the **+ Comment** button that appears
4. Type your instruction (e.g. "make this shorter", "fix the grammar", "turn this into a list")
5. Hit **Apply** — Claude will rewrite the file instantly

## Features

- Renders markdown as you'd see it in a final document
- No raw syntax, just clean readable text
- Changes apply immediately and the view refreshes automatically
- Works on any folder of markdown files

## Tips

You can be as specific or vague as you want with your instructions. Try things like:

- "Rewrite this in a more casual tone"
- "Add an example here"
- "Make this a bullet list"
- "This paragraph is confusing, clarify it"
